 Notes
 
  AutoCompleteWord property: When it uses the property AutoCompleteWord and the list is shown,
                             the element was not located in the list to make quicker the
                             search of the text to complete.  
  SeparatorLine parameter:   When it uses the parameter SeparatorLine it is not applicable
                             neither to the first element of the list (provided a single
                             element exists) neither to the last one.  
  MouseIcon parameter:       It is not applied to inactive elements of the list.
